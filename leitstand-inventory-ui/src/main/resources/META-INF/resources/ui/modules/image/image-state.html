A new image is an image that has not yet been approved to be eligible for a deployment. The release image is deployed by default. A candidate image is intended to become the new default image, once the candidate has passed all necessary tests. The existing default image is superseded by the new default image. A revoked image must not be used any longer because it contains severe errors. New Newly registered image Candidate Candidate for the next release image Release The image to be installed by default Superseded A former release image that has been superseded by a newer image version Revoked The image must not be used anymore.
